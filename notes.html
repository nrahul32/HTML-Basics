Links

Links can be added using 'a' anchor tag and 'href' attribute. To open it in a new tab, use the 'target' attribute whose values can be:
_blank for new tab
_self for its own window (default)
_parent for opening in a document frame if there was one
_top opens in full body of window

HTML Entities

When we want to show the html elements with the tags like <div>'s, they don't show up. So we can use HTML Entities. They start with & and end with ;
Use lt to represent < and gt for >
Some entities:

  • © : Copyright symbol
  • • : Bullet symbol
  • & : AND symbol
  • < : Less than symbol
  • > : Greater than symbol
  • → : Right arrow symbol
  • ← : Left arrow symbol
  • » : Double right chevrons
  • « : Double left chevrons

Semantics

Divs were used for most of the cases. HTML5 introduced some more elements for specific uses, so that the code is more understandable:

  1. nav - for navigation bar
  2. header - at the top
  3. footer - the bottom bar
  4. article - for articles
  5. aside - side bar
  6. details - details like copyright info
  7. main - main content section
  8. time - time fields in the page
  9. mark - highlighted text
